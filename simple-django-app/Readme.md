
# Getting Started with simple-django-app Project

## Github link to the project

<https://github.com/deparkes/simple-django-app>

## Clone the Project

1. Navigate to the cloned project directory

```bash
   cd simple-django-app
```

2. Clone the 'simple-django-app' project using the following command

```bash
   git clone https://github.com/deparkes/simple-django-app.git
```

## Build Docker Image

1. Verify the presence of the Dockerfile. If it's present in the root directory, there is no need to copy it.

2. Build the Docker image named 'django-app' using the following command

```bash
   docker build -t django-app .
```

For more information and usage guidelines, refer to the project's documentation and codebase.
